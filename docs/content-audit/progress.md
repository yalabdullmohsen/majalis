## قيود التنفيذ المتّبعة هذه الجلسة
- Agent واحد فقط، بلا subagents.
- `pnpm run build` الكامل يتطلب `PORT=3000 BASE_PATH=/` بيئياً (حزم تسويقية غير مرتبطة تتطلبه إلزامياً حتى للبناء الثابت).
- عند تشغيل أي سكربت مزامنة: **راجع `git diff --stat` دائماً** وأعد أي ملف غير مرتبط بالدفعة الحالية لحالته الأصلية (`git checkout --`) قبل commit.
- عند حذف سجل مكرر: تحقّق دائماً من عدم وجود مرجع خارجي بمعرّفه (`grep -rn '"the-id"'`) قبل الحذف، واحرص على مطابقة حدود الكائن كاملة (بما فيها الفاصلة والقوس التالي) لتفادي كسر بنية الملف.
- لا دمج ولا push إلى main — فقط `automation/content`.
