## قيود التنفيذ المتّبعة هذه الجلسة
- Agent واحد فقط، بلا subagents.
- شُغِّل `pnpm run build` كاملاً مرتين (بوابة الجودة الإلزامية في `scripts/commit-and-push-branch.sh`) — يتطلب `PORT=3000 BASE_PATH=/` بيئياً بسبب حزم تسويقية غير مرتبطة (`mockup-sandbox`, `majalis-pitch`, `majalis-promo`) تتطلب PORT إلزامياً حتى للبناء الثابت.
- عند تشغيل `sync-seo-data.ts` أو أي سكربت مزامنة: **راجع `git diff --stat` دائماً قبل الإضافة للـcommit** — ثبت عملياً أنه قد يُنتج ملفات تخص نطاقاً أوسع من المقصود (كتشخيص b006).
- لا دمج ولا push إلى main — فقط `automation/content`.
