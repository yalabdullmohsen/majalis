## أول عنصر للبدء منه عند الاستئناف
مسح التكرار **مغلق نهائياً** (b031). تأكَّدت جودة عالية في: السيرة، القواعد الفقهية، الإعجاز العلمي، التاريخ الإسلامي (b030)، وعلوم القرآن/المتشابهات (b032). العمل المتبقي هو **`b010`**: مواصلة عيّنات المراجعة العلمية اليدوية العميقة لبقية الأقسام غير المُختبَرة بعد: **التفسير التفصيلي**، **مصطلح الحديث** (المحتوى الفعلي وليس فقط المصطلحات المُدقَّقة سابقاً)، **العقيدة** (غير المصطلحات)، **الصحابة** (بقية حقوله fadl/legacy واحداً واحداً)، **الأخلاق** (المحتوى الفعلي بعد تنظيف التكرار)، **اللغة العربية والبلاغة**. اختر صفحة واحدة محورية لكل دفعة واقرأها كاملة (لا فحصاً آلياً فقط) بحثاً عن جزم غير موثق أو إسرائيليات أو أخطاء تاريخية، على غرار عيّنة هذه الدفعة. `b011`، `b014`، `b018` معلَّقة (needs_source) — لا تُعاد فتحها آلياً بلا مصدر جديد.

## قيود التنفيذ المتّبعة هذه الجلسة
- Agent واحد فقط، بلا subagents.
- `pnpm run build` الكامل يتطلب `PORT=3000 BASE_PATH=/` بيئياً (حزم تسويقية غير مرتبطة تتطلبه إلزامياً حتى للبناء الثابت).
- عند تشغيل أي سكربت مزامنة (`sync-seo-data.ts` وغيره): **راجع `git diff --stat` دائماً** قبل الإضافة للـcommit وأعد للحالة الأصلية أي ملف غير مرتبط بالدفعة الحالية (`git checkout --`) — ثبت عملياً أنها قد تُنتج انحرافاً واسعاً (b006) أو تكشف ملفات مولَّدة قديمة غير مرتبطة (rulings-encyclopedia chunks, seo-prerender/fiqh-council/*) لا يجب تضمينها.
- لا دمج ولا push إلى main — فقط `automation/content`.
