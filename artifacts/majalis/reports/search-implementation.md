## 2. منطق التطبيع العربي

### خطوات التطبيع (الترتيب مهم)

1. **إزالة التشكيل** (U+064B–U+065F، U+0670)
2. **إزالة علامات الوقف القرآنية** (U+06D6–U+06ED)
3. **إزالة الكشيدة** (ـ)
4. **توحيد صور الألف**: أ إ آ ٱ ← ا
5. **توحيد الهمزات**: ؤ ← و، ئ ← ي
6. **توحيد التاء المربوطة**: ة ← ه
7. **توحيد الألف المقصورة**: ى ← ي
8. **إزالة علامات الترقيم** والمسافات الزائدة

### مثال حي

| نص الإدخال | بعد التطبيع |
|------------|-------------|
| `بِسْمِ ٱللَّهِ ٱلرَّحْمَٰنِ ٱلرَّحِيمِ` | `بسم الله الرحمان الرحيم` |
| `أذكار الصباح` | `اذكار الصباح` |
| `الصَّلَاةُ` | `الصلاه` |
| `قُرْآنٌ كَرِيمٌ` | `قران كريم` |

## 3. أمثلة على الاستعلامات والنتائج

### 3.1 بحث "اذكار الصباح"

```
GET /api/search?q=اذكار الصباح&types=lessons,adhkar

النتائج:
{
  "ok": true,
  "query": "اذكار الصباح",
  "normalized": "اذكار الصباح",
  "results": [
    { "type": "lesson", "title": "أذكار الصباح والمساء", "status": "approved" },
    { "type": "adhkar", "title": "أذكار الصباح", "verified": true }
  ]
}
```

> ملاحظة: "اذكار" (بدون همزة) يجد "أذكار" (بهمزة) بفضل توحيد الألف.

### 3.2 بحث "الرحمن"

```
GET /api/search?q=الرحمن&types=quran

النتائج:
{
  "results": [
    {
      "type": "quran",
      "title": "بِسْمِ ٱللَّهِ ٱلرَّحْمَٰنِ ٱلرَّحِيمِ",   ← verbatim
      "verbatim": true,
      "href": "https://quran.com/1/1"
    }
  ]
}
```

> النص القرآني المعروض للمستخدم هو النص الأصلي الحرفي من واجهة برمجة AlQuran Cloud.

### 3.3 بحث "صلاه" (بتاء مربوطة خاطئة)

```
GET /api/search?q=صلاه

النتائج: يعيد نتائج "صلاة" لأن ة و ه متطابقتان بعد التطبيع.
```

## 5. ضمان عرض النص القرآني حرفياً

### القاعدة الصارمة
> النص المعروض للمستخدم — خاصةً آيات القرآن الكريم — يُعرض **حرفياً بتشكيله الكامل** دون أي تعديل. التطبيع للفهرسة فقط.

### كيفية التطبيق

1. **مصدر النص القرآني**: واجهة `api.alquran.cloud/v1/search` — لا نصوص قرآنية محلية.
2. **في API**: النتائج من نوع `quran` تحمل `verbatim: true` وتُرسل `ayah_text` الأصلي كـ `title`.
3. **في الواجهة**: `QuranAyahResultRow` يعرض النص بـ `font-family: "Amiri Quran"` وخاصية `dir="rtl" lang="ar"`.
4. **في CSS**: `.search-result-row--quran .search-result-title` يستخدم `var(--font-quran, "Amiri Quran")`.
5. **في قاعدة البيانات**: `quran_search_index.ayah_text` يُخزَّن الأصلي بينما `search_text` هو النسخة المطبَّعة.

### اختبار التحقق

```typescript
// من scripts/verify-search.ts
const isVerbatim = row.ayah_text !== row.search_text;
assert(isVerbatim, "ayah_text ≠ search_text (النص الأصلي غير مطبَّع)");
```
