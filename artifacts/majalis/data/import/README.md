# ملفات الاستيراد الجماعي — المجلس العلمي

جاهزة للنسخ في **لوحة التحكم** → **⇪ استيراد جماعي**.

## الترتيب (مهم)

| # | الملف | القسم في `/admin` | العدد |
|---|-------|-------------------|-------|
| 1 | `01-sheikhs.json` | المشايخ | 4 |
| 2 | `02-kuwait-lessons.json` | الدروس | 8 |
| 3 | `03-fawaid.json` | الفوائد | 10 |
| 4 | `04-qa.json` | الأسئلة | 10 |

**استورد المشايخ أولًا** حتى يُربط كل درس بالشيخ تلقائيًا عبر `sheikh_name`.

## خطوات الاستيراد

1. سجّل دخولك كمسؤول: https://www.majlisilm.com/admin
2. افتح القسم (مثل «المشايخ»).
3. اضغط **⇪ استيراد جماعي**.
4. افتح الملف من هذا المجلد، انسخ المحتوى بالكامل، والصقه.
5. اضغط **بدء الاستيراد**.
6. كرّر للملف التالي بالترتيب.

## بعد الاستيراد

- **صور المشايخ:** الملف يضع روابط SVG مؤقتة. لصور حقيقية: عدّل كل شيخ → «رفع صورة».
- **دروس الكويت:** تأكد أن الحالة **معتمد** (موجودة مسبقًا في الملف).
- **الأسئلة:** التصنيفات تُطابق أسماء قاعدة البيانات (`أحكام شرعية`، `السيرة النبوية`…).
- **المساعد العلمي:** أضف `ANTHROPIC_API_KEY` في Vercel → مشروع API → Redeploy.

## إعادة توليد الملفات

```bash
cd artifacts/majalis
node scripts/generate-import-files.mjs
```

## تعديل يدوي

عدّل `scripts/generate-import-files.mjs` أو عدّل ملفات JSON مباشرة قبل الاستيراد.
