# نظام التصميم — مؤشّر، ليس مصدرًا موازيًا

**لا تنشئ نظام تصميم جديدًا هنا.** المصدر الوحيد الموثوق للتوكِنز
الفعلية هو `src/styles/modern-2026.css`. هذا الملف مؤشّر فقط:

- الألوان + التباين: `../design-tokens.md` (أُعيد كتابته 2026-07-24 ليطابق `modern-2026.css` فعليًا).
- المكوّنات/التباعد/الظلال القديمة (v2): `../design-system.md` — **موسوم بتنبيه تعارض غير مُراجَع كاملًا بعد**، راجع `backlog.json` → `docs-design-system-v2-reconcile`.
- الجرد الآلي للمكوّنات المشتركة والمسارات: `inventory.json`.

عند إضافة أي مكوّن أو صفحة جديدة: استخدم متغيّرات `modern-2026.css`
(`--m26-*`, `--txt-*`) مباشرة، لا قيمًا حرفية، ولا أسماء `--ds-*`/`--majalis-*`
القديمة إلا إن تحقّقت أنها لا تزال حيّة فعليًا في الكود الحالي.
