# تدقيق خط الأساس — قبل إعادة الهيكلة

**التاريخ:** 2026-07-10
**الغرض:** تسجيل الحالة الراهنة قبل تطبيق تحسينات P0-P3

---

## 1. مشكلات SEO (P0)

| المشكلة | التفاصيل |
|---|---|
| HTML فارغ للزواحف | الموقع SPA كامل — المتصفح يُنفّذ JS لعرض المحتوى. الزواحف تحصل على shell فارغة |
| og:locale خاطئ | `ar_AR` بدلاً من `ar_KW` |
| meta مشتركة لجميع الصفحات | الصفحة الرئيسية تُعيد نفس العنوان للدروس والفتاوى والأحكام |
| الملفات المُعالَجة مسبقًا `seo-prerender/` | 57 مسارًا مُعدًّا لكنها لا تُدمج مع JS الـ SPA |
| لا يوجد ملف `middleware.ts` للـ Edge | لا يوجد bot-detection على مستوى Edge |
| JSON-LD جزئي في صفحات التفاصيل | صفحات الدروس/الفتاوى/الأحكام لا تُدرج JSON-LD خاصًا بها |

## 2. هيكل التنقل الحالي (P1)

- التنقل السفلي: 5 تبويبات (الرئيسية، المصحف، الأذكار، الصلاة، المزيد)
- القائمة الجانبية: قائمة مسطّحة غير مُصنّفة بـ 80+ رابط
- لا يوجد تصنيف واضح للمستخدم الجديد

## 3. مشكلات الرئيسية (P1)

- الرئيسية مكدّسة بأقسام (16+ مكوّن)
- لا يوجد عنوان/رسالة واضحة للمنصة أعلى الصفحة
- مربع البحث الموحّد ليس بارزًا بما يكفي
- زائر جديد يحتاج أكثر من نقرتين للوصول لأول درس

## 4. الصفحات الناقصة/placeholder (P1)

- `/study-room` — تجريبي
- `/family` — تجريبي
- `/car-mode` — تجريبي
- `/mosque-mode` — تجريبي
- `/vault` — تجريبي
- `/transcribe` — تجريبي
- `/academic-research` — غير مكتمل
- `/flashcards` — تجريبي

## 5. قياسات الأداء (P3) — تقديرية

- حزم Vite الرئيسية: `vendor`, `supabase`, `radix`, `icons`, `charts` — code splitting موجود
- لا يوجد WebP/AVIF تحويل تلقائي
- خطوط: 8+ عائلات مُحمَّلة (Almarai, El Messiri, Cairo, Tajawal, Amiri, Scheherazade, Amiri Quran, Reem Kufi)
- prefetch للخطوط موجود في index.html

## 6. مسارات تحتاج JSON-LD

| النوع | Schema |
|---|---|
| دروس `/lessons/:id` | EducationEvent / Course |
| فتاوى `/fatwa/:id` | Article (Q&A) |
| أحكام `/rulings/:id` | Article |
| دورات `/annual-courses/:id` | Course |
| مجمع فقهي `/fiqh-council/:slug` | Article |
| صفحات الأنبياء `/prophets/:slug` | Article |
