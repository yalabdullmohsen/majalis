# نظام الألوان المعتمد — مجالس

## الألوان الأساسية

| المتغير | القيمة | الاستخدام |
|---------|--------|-----------|
| `--msk-gold` / `--ds-emerald` | `#0E6E52` | اللون الأساسي — أخضر زمردي |
| `--msk-gold-deep` | `#0A5040` | الأخضر الداكن للتأكيد |
| `--msk-canvas` | `#FAF8F5` | الخلفية الرئيسية |
| `--msk-canvas-1` | `#FFFFFF` | سطح البطاقات |
| `--msk-text` | `#1C1A18` | النص الأساسي |
| `--msk-border` | `#E0DBD5` | الحدود |

## ممنوع منعاً باتاً

- ❌ أي قيمة hex بنية: `#92400E`, `#B45309`, `#a16207`, `#c8a24a`
- ❌ فئات Tailwind: `amber-*`, `orange-8*`, `orange-9*`, `yellow-7*`, `yellow-8*`, `yellow-9*`
- ❌ أي CSS variable باسم يتضمن `gold` إلا إذا كانت قيمتها `#0E6E52` أو متغير منه

## مسموح استثنائياً

- ✅ الذهبي الفاتح جداً كتفصيل صغير: `rgba(14,110,82,0.1)` فقط
- ✅ `--majalis-warning: #B45309` للتحذيرات الوظيفية (أخطاء، تنبيهات) فقط — لا للزخرفة
- ✅ ألوان الوضع المظلم `html[data-theme="dark"]` لها قواعدها الخاصة

## الوضع المظلم

يستخدم: ذهبي `#D9AE5C` على خلفية داكنة `#171C2E` — هذا مقبول في dark mode فقط.
