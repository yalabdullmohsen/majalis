1. اعمل من `/Users/alabdullmohsen/majalis-content-fill` (فرع `majalis-content-fill`).
2. `git fetch origin majalis-content-fill && git pull --ff-only` قبل أي تعديل.
3. بعد التعديل والفحوصات: `git add` للملفات ذات الصلة فقط (لا `git add -A`)، commit برسالة تلخّص التغيير فعليًا.
4. `git push origin majalis-content-fill`.
5. انتقل إلى `/Users/alabdullmohsen/majalis-task-2` (فرع `main`): `git fetch origin main && git pull --ff-only`، ثم `git merge --no-edit majalis-content-fill`.
6. شغّل الفحوصة الموجّهة + `pnpm run build` هناك أيضًا، وأعد `git checkout HEAD --` لأي ملفات ضجيج مولَّدة (rulings-encyclopedia، feed.xml، sitemap.xml، content-counts.json إن لم يتغيّر رقم حقيقي).
7. **قبل الدفع**: `git fetch origin main` مرة أخيرة وتحقّق `git log --oneline HEAD..origin/main` — إن ظهرت التزامات جديدة (نافذة أخرى تعمل بالتوازي، هذا متوقَّع وليس خطأ)، `git merge --no-edit origin/main` وأعد الفحص، **لا تدفع أبدًا بـforce**.
8. `git push origin main`.
9. **ممنوع** لمس فرع `production` إطلاقًا أو الدفع إليه مباشرة.
